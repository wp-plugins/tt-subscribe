== Description ==

This plugin is a simplified subscription box that is totally customizable. It currently includes facebook, twitter, google plus and feedburner. Other social media links will be included soon. Created by Thesabel Tuto.

**NEW!!!** Updates:
1. Shortcodes: `[tt_subscribe]`
2. Widget: `TT-Subscribe`
3. Fixed layout on content area
4. Customizable header color

Checkout this plugin's website: [http://ttplugins.wordpress.com/](http://ttplugins.wordpress.com/)

**DISCLAIMER AREA**

Facebook Information:
This plugin uses the current and simplified LIKE BUTTON of facebook.com. Security issues are handled by facebook. This plugin will not need any other  private information regarding your facebook account.

Twitter Information:
This plugin uses the current and simplified FOLLOW ME BUTTON of twitter.com. Security issues are handled by twitter. This plugin will not need any other  private information regarding your twitter account.

Google Plus Information:
This plugin uses the current and simplified PROFILE BADGE of google.com. Security issues are handled by google. This plugin will not need any other  private information regarding your google account.

Feedburner Information:
This plugin uses simple link of your feedburner.com. Security issues are handled by feedburner. This plugin will not need any other private information regarding your feedburner account.

== Installation ==

1. Upload `tt-subscribe` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. **NEW!!!** There are **3 ways** to include TT-Subscribe to your WordPress Site! Here are your Options: 
* Place `<?php do_action('tt_subscribe'); ?>` in your templates
* **NEW!!!** Add shortcode `[tt_subscribe]` to your Posts or Pages.
* **NEW!!!** Drag and drop widget called `TT-Subscribe` to your Theme.

== Frequently asked questions ==

* Where can I change the settings?
* A: You can change the settings in the Dashboard -> TT Subscribe menu.
